# Usage
<!-- usage -->
```sh-session
$ npm install -g @commercelayer/commercelayer-cli
$ commercelayer COMMAND
running command...
$ commercelayer (-v|--version|version)
@commercelayer/commercelayer-cli/0.6.1 darwin-x64 node-v15.13.0
$ commercelayer --help [COMMAND]
USAGE
  $ commercelayer COMMAND
...
```
