### `commercelayer update`

Update Commerce Layer CLI.

```sh-session
USAGE
  $ commercelayer update [-v <value>]

FLAGS
  -v, --version=<value>  Update CLI to a specific version or tag

DESCRIPTION
  Update Commerce Layer CLI

EXAMPLES
  $ commercelayer update --version=<version-or-tag>
```
